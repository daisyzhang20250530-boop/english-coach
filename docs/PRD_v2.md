# PRD：职场英语刻意练习教练 v2.0

> 文档状态：定位重构版 ｜ 日期：2026-06-10 ｜ 作者：你 + Claude
> 相比 v1 的根本变化：定位从「救急解读工具」改为「刻意练习训练引擎」。救急不是本产品职责。

## 1. 产品定位（What & Why）

**一句话定位**：一个基于刻意练习（Deliberate Practice）方法论的职场英语训练引擎，用你真实工作中的弱点生成针对性训练，每天 15 分钟，把商务英语的句式结构内化成直觉。

**用户画像**：雅思 7（口语 6 / 写作 6.5）的职场人。已越过"天真练习"（大量接触即可进步）的阶段，平台期突破必须依赖有目标、有反馈、走出舒适区的刻意练习。

**要解决的三个能力缺口**：

| 工作场景痛点 | 能力本质（心理表征视角） | 对应训练线 |
|---|---|---|
| PPT 材料要读好几遍 | 大脑中缺少「商务长难句结构模板」，无法一眼定位主干 | 解码线 |
| 写出来不确定是否正确/formal | 缺少「语域档位」的判断标准和 formal 句式库 | 语域线 |
| 会议中说不成完整句 | 缺少可即时调用的「职场表达句式库」，实时组句负荷过高 | 组句线 |

## 2. 设计原则：刻意练习四要素如何落地

本节是全产品的灵魂，所有功能设计必须能回答"符合哪一条"。

**原则一：明确而具体的目标。** 禁止"提高英语"这类模糊目标。每条训练线由一串可量化的微目标组成（见第 4 节），用户任何时刻都知道自己在练什么、当前水平、距下一级多远。

**原则二：走出舒适区（难度自适应）。** 每种题型有难度阶梯（句长、时限、黑话密度、语域跨度）。连续正确 → 升档；连续卡住 → 降档。系统始终把用户压在"踮脚够得着"区间。**绝不允许用户停留在能轻松答对的难度刷成就感。**

**原则三：即时且具体的反馈（教练，不是答案机）。** 反馈分两步：先指出问题位置和类型，引导用户自己修正一轮；用户修不动了，才展示参考版本 + 逐点解释"为什么这样更好"。AI 的角色是书中的教练，核心动作是"指出你自己看不到的问题"。

**原则四：构建心理表征。** 训练单位是**句式结构和表达模式**，不是孤立单词。同一结构会以不同内容反复出现（结构迁移训练），直到用户对该结构的处理从"分析"变成"直觉"。

**辅助原则：短时高专注。** 每日 1 个 session，时长由用户当天选择（15/30/45/60 分钟），以 15 分钟训练块为基本单元，块间强制休息，倒计时营造适度压力。不做无限刷题模式——疲劳状态下的练习是天真练习。

## 3. 核心循环（产品骨架）

```
┌──────────────────────────────────────────────────┐
│  素材输入（可选）：贴入真实工作材料（PPT文字/IM/邮件）   │
│       ↓ AI 提取：难句结构、商务表达、你的语域偏差        │
│  弱点档案：错误类型统计 + 各微目标的当前等级             │
│       ↓ 引擎依据弱点编排                              │
│  每日训练 Session（15分钟，三线混编 6-8 题）            │
│       ↓ 先答题 → 教练式反馈 → 自我修正 → 参考版本       │
│  数据回写：等级升降、错误归档、掌握度更新                │
│       └────────────── 循环 ──────────────────────┘
```

真实材料不是必需的（内置商务语料兜底），但贴入越多，训练越贴合你的工作语境。

## 4. 三条训练线详细设计

### 4.0 摸底测试（首次使用必经，约 5 分钟）

**目的**：为三条线确定起始难度等级，避免从 L1 爬升期过于简单（舒适区练习无效）。

- **结构**：共 9 题，每条线 3 题，复用正式训练题型（解码线：限时主干提取；语域线：档位判断 + 单句改写；组句线：限时组句）。
- **自适应规则**：每条线从 L2 难度起测，答对升一级出题、答错降一级，3 题后锁定起始等级（允许 ±1 级误差，由日常升降档机制继续校准）。
- **产出**：三条线的起始等级 + 初始错误归类，构成第一份弱点档案；首个 session 的出题配比据此生成。
- **附带价值**：摸底即新手引导，做完即理解产品玩法。
- **规则**：首次不可跳过；之后可在设置中每 4 周重测一次，校准训练等级与真实能力。

### 4.1 解码线 —— 目标：商务材料一遍读懂

**微目标阶梯**（示例，可调）：
- L1：20 词以内单句，10 秒内选出正确主干（主谓宾）
- L2：30 词含名词化/被动的句子，10 秒内选出主干
- L3：40 词以上多层修饰长句，15 秒内选出主干
- L4：一段 slide 文字（3-5 句），30 秒内用一句中文概括主旨
- L5：同 L4，但含行业缩写与黑话，且概括需用英文

**题型**：
1. **限时主干提取**：展示难句 + 倒计时，从选项中选出句子主干；错了→AI 标出干扰你的修饰成分，解释该结构的识别套路。
2. **限时主旨概括**：展示一段商务文字 + 倒计时，用户输入一句话主旨；AI 对比评分（抓没抓住核心动作和结论）。
3. **结构迁移**：刚练过的句式结构，换一个内容再来一句（巩固心理表征的关键题型）。

### 4.2 语域线 —— 目标：产出正确且语域得体

**微目标阶梯**：
- L1：给定句子判断语域档位（casual / neutral / formal），正确率 ≥ 90%
- L2：casual → formal 单句改写，AI 评分 ≥ 80
- L3：修复含典型中式英语/搭配错误的句子
- L4：给定中文要点 + 场景（PPT/IM/邮件），限时写出语域得体的英文
- L5：同 L4 且一次通过（无需修正轮）

**题型**：
1. **语域判断题**：快速判断句子的正式度档位，建立"档位感"。
2. **语域转换题**：把 casual 句改写成 PPT 可用的 formal 版；先自己写 → AI 指出残留的 casual 痕迹 → 自己再改 → 看参考版。
3. **纠错题（个性化核心）**：直接从你的错误本出题——你上周被批改出的错误类型，本周变成新句子里的考题，直到该类型错误率收敛。

### 4.3 组句线 —— 目标：会议表达的"提取速度"（文字版先行）

**微目标阶梯**：
- L1：给中文意思 + 句式提示，无时限写出英文
- L2：撤掉句式提示，30 秒时限
- L3：20 秒时限，且要求使用指定的职场高频句式（如 "I'd suggest we..." / "My concern is..."）
- L4：连续 3 句组成完整发言（提出问题→给出建议→请求反馈），60 秒
- L5：同 L4，15 秒/句（接近实时会议压力）

**题型**：
1. **限时组句**：会议场景中文意思 + 倒计时 → 打出英文 → AI 给出 native 版本并拆解差距。
2. **句式填装**：给定职场句式骨架，限时填入当前场景内容——目的是把高频句式练成肌肉记忆，会议时直接调用。
3. **场景接龙**：AI 模拟会议抛出一句话（如老板提问），你限时打出回应。

> 语音版（听+说）为二期，但 L1-L5 的设计将直接复用，仅把输入输出换成语音。

### 4.4 弱点档案（替代 v1 的"学习库"）

- **等级面板**：三条线 × 各微目标的当前等级与进度，一眼看到"我在哪、下一步是什么"（原则一的载体）。
- **错误图谱**：错误按类型聚合（冠词/时态/搭配/语域/中式直译/结构误读），显示各类型的出现频率与收敛趋势。
- **表达卡池**：从真实材料和训练中积累的商务表达与句式，状态分为「见过 → 练过 → 已掌握」，"已掌握"的判定是在组句题中主动正确使用过 ≥ 2 次。

## 5. 每日 Session 的编排逻辑

**时长可选**：每次开始训练前，由用户根据当天情况选择 15 / 30 / 45 / 60 分钟。

| 时长 | 结构 | 专注保护机制 |
|---|---|---|
| 15 min | 1 个训练块（6-8 题） | 标准刻意练习单元 |
| 30 min | 2 个训练块 | 块间强制休息 2 分钟 |
| 45 min | 3 个训练块 | 块间休息 2-3 分钟 |
| 60 min | 4 个训练块 | 同上 + 疲劳监测：第 3 块起若正确率明显下滑，建议提前结束（疲劳练习会退化为天真练习） |

多块 session 采用**交错编排**：块与块之间切换训练线（如 块1解码 → 块2语域 → 块3组句），利用 interleaving 效应提升记忆保持并降低疲劳。

单块内的编排规则：

1. 6-8 题，三线混编（默认配比 = 弱点权重：错误图谱中占比高的类型多出题）。
2. 每个 session 固定含 1 道「复仇题」：你最近错过的题型/结构换内容重出。
3. 每个 session 固定含 1 道「迁移题」：上个 session 练过的结构换场景再现。
4. 结束页只展示三件事：今天练了什么微目标、哪里进步了、明天会练什么——强化目标感，不做花哨的打卡动画。

**自由加练**：每日 session 之外允许随时加练任意一条训练线（连续出题模式）。防疲劳机制：每 10 题检查正确率，若较该线日常水平明显下滑则提示停止（疲劳练习会固化错误）。加练数据同样回写弱点档案。

**语言策略**：界面与反馈默认全英文（反馈本身即阅读训练）。看不懂时三级阶梯：英文反馈 → 点击 "Simpler" 以更简单的英语重述 → 中文兜底（入口较深，防止依赖）。

## 6. 成功指标

| 维度 | 指标 | 目标 |
|---|---|---|
| 能力（核心） | 解码线 L4 主旨概括正确率 | 4 周内从基线提升至 80% |
| 能力 | 语域线错误图谱中 top1 错误类型的出现率 | 4 周内下降 50% |
| 能力 | 组句线达到 L3 | 6 周内 |
| 行为 | 工作日 session 完成率 | ≥ 70% |
| 主观 | 你自评"读 PPT 需要的遍数" | 从 2-3 遍降到 1 遍 |

## 7. 非目标

- ❌ 救急式解读/代写：**明确不做**。贴入材料只用于生成训练素材，不提供"直接给我答案"的快捷通道。
- ❌ 语音听说训练 → 二期（设计已预留）。
- ❌ 完整 SRS 算法、多用户、社交、打卡奖励体系。
- ❌ 上传解析 PPT 文件（先支持纯文本粘贴）。
