# 职场英语教练 Workplace English Coach

基于刻意练习方法论的职场英语训练引擎：解码（读）/ 语域（写）/ 组句（说前置）三线训练，难度自适应、SRS 间隔复习、表达卡池、真实工作素材出题。

- **在线使用**：GitHub Pages 部署的 `index.html`（首次使用需粘贴自己的 DeepSeek API 密钥，密钥只存本机浏览器）
- **产品文档**：[docs/PRD_v2.md](docs/PRD_v2.md)
- **开发文档**：[docs/开发文档.md](docs/开发文档.md)（架构、构建、发布流程都在这里）
- **源代码**：[app/](app/)

⚠️ 本仓库不含任何密钥。`app/.env.local` 与令牌文件永不上传。
