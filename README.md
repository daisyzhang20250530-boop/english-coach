# 职场英语教练 · Workplace English Coach

> 一个基于**刻意练习**方法论的职场英语训练引擎。不是又一个背单词 App——它用你真实工作中的弱点生成针对性训练，每天 15 分钟，把商务英语的句式结构练成直觉。

**🔗 直接使用：https://daisyzhang20250530-boop.github.io/english-coach/**

## 适合谁

有一定英语基础（雅思 6-7 / CET-6 以上）、在外企或英语工作环境的职场人，典型痛点：

- 📊 PPT 和邮件里的长难句要读好几遍才懂
- ✍️ 写出来的英文不确定对不对、语气合不合适
- 💬 会议上组织不出完整的句子，说出口总是碎片

如果你是零基础背单词阶段，这个工具不适合你——它默认你"认识词但用不流利"。

## 三条训练线

| 训练线 | 治什么 | 怎么练 |
|---|---|---|
| **Decode 解码** | 长难句读得慢 | 倒计时内从长句里抓出主干（谁做了什么），答完在原句上标注主谓宾；11 种商务句法结构逐个击破 |
| **Register 语域** | 语气拿不准 | 给你一个沟通意图（拍板通知 / 试探提议 / 委婉反对…），把草稿润色成语气匹配的版本 |
| **Build 组句** | 开会说不成句 | 中文意思+场景+倒计时，限时打出英文；含"老板抛来一句话你来接"的接龙题 |

## 科学机制（为什么有效）

- **难度自适应**：连对升级、连错降级，永远把你压在"踮脚够得着"的区间
- **教练式反馈**：先指出问题让你自己改，改不动才给参考答案——不是答案机
- **艾宾浩斯间隔复习**：你的弱点表达、答错的题，按 1→3→7→16→35 天的遗忘曲线自动安排重考
- **表达卡池**：练习中遇到的更地道说法自动存卡，在新场景里主动用对 2 次才算掌握
- **真实素材**：贴入你自己的 PPT/邮件/IM 文字，一半的题目直接来自你的工作语境
- **语气速查卡**：6 种沟通意图 × 语气档位的参考表，内置在应用里

## 快速开始（5 分钟）

1. **打开应用**：上面的链接，电脑手机浏览器都行
2. **准备一把 AI 钥匙**：应用的出题和批改由 DeepSeek 完成，需要你自己的 API 密钥
   - 去 https://platform.deepseek.com 手机号注册 → 充值 10 元（够练几个月，一次 15 分钟训练约几分钱）→ API keys → 创建并复制 `sk-` 开头的密钥
   - 首次打开应用时粘贴，密钥**只保存在你自己的浏览器里**，不会上传
3. **做摸底测试**：9 题约 5 分钟，定出你三条线的起始难度
4. **每天一个 15 分钟 session**：高频短时 > 周末突击，坚持是唯一的前提

## 多设备同步（可选）

想在公司和家里共享进度：注册一个 GitHub 账号 → [创建令牌](https://github.com/settings/tokens/new)（只勾选 **gist** 权限）→ 在应用主页点"开启云同步"粘贴。进度会存进你自己账号的私密 Gist，每台设备设置一次，之后全自动。

## 隐私与费用

- 密钥、进度只存在**你自己的**浏览器和 GitHub 账号里，本项目没有服务器，作者收不到你的任何数据
- 唯一的持续成本是你自己的 DeepSeek API 用量（很便宜）
- ⚠️ "我的素材"里贴入的文字会发送给 DeepSeek 用于出题——涉及公司机密的名称和数字请先手动替换
